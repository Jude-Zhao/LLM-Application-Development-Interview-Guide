# 图论

## 1. 核心思想

图论题的本质是：

- 点与点之间存在连接关系
- 你需要在这种连接关系上做搜索、建模或统计

在 LeetCode 里，图论不一定会直接给你“图”这个词，它经常伪装成：

- 课程依赖
- 岛屿连通
- 单词变换
- 复制带随机指针的结构
- 从一个状态变到另一个状态

所以看到“节点之间可以互相到达”“状态之间可以转移”时，就要想到图。

图论题最常见的三类方法是：

- `DFS`：一条路走到底，适合连通块、回溯式遍历
- `BFS`：一层一层扩展，适合最短步数
- 并查集 / 拓扑排序：适合连通性维护、依赖关系判断

## 2. 基础模板

### 2.1 DFS 遍历模板

```python
def dfs(graph, node, visited):
    if node in visited:
        return

    visited.add(node)

    for nei in graph[node]:
        dfs(graph, nei, visited)
```

适用场景：

- 连通块统计
- 路径搜索
- 岛屿问题

### 2.2 BFS 最短步数模板

```python
from collections import deque


def bfs(graph, start):
    queue = deque([(start, 0)])
    visited = {start}

    while queue:
        node, step = queue.popleft()

        if node == target:
            return step

        for nei in graph[node]:
            if nei not in visited:
                visited.add(nei)
                queue.append((nei, step + 1))

```

适用场景：

- 无权图最短路径
- 最少转换次数
- 最少步数

### 2.3 网格图 DFS 模板

```python
def dfs(grid, i, j):
    m, n = len(grid), len(grid[0])

    if i < 0 or i >= m or j < 0 or j >= n:
        return
    if grid[i][j] != "1":
        return

    grid[i][j] = "0"

    for dx, dy in [(1, 0), (-1, 0), (0, 1), (0, -1)]:
        dfs(grid, i + dx, j + dy)
```

适用场景：

- 岛屿数量
- 封闭区域
- 连通区域染色

### 2.4 拓扑排序模板

```python
from collections import deque


def topo_sort(numCourses, prerequisites):
    graph = [[] for _ in range(numCourses)]
    indegree = [0] * numCourses

    for a, b in prerequisites:
        graph[b].append(a)
        indegree[a] += 1

    queue = deque(i for i in range(numCourses) if indegree[i] == 0)
    order = []

    while queue:
        cur = queue.popleft()
        order.append(cur)

        for nei in graph[cur]:
            indegree[nei] -= 1
            if indegree[nei] == 0:
                queue.append(nei)

    return order if len(order) == numCourses else []
```

适用场景：

- 课程表
- 依赖安装顺序
- 判断图中是否有环

## 3. 写图论题时最容易错的点

### 3.1 没先想清楚“节点”和“边”是什么

很多题做不出来，不是不会搜，而是没建好图。

例如：

- 岛屿题：每个格子是节点，相邻陆地之间有边
- 课程表：课程是节点，先修关系是有向边
- 单词接龙：单词是节点，只差一个字符的单词之间有边

### 3.2 `visited` 标记时机不对

在 BFS 中，通常是：

- 入队时就标记访问过

否则同一个节点可能被重复入队很多次。

### 3.3 DFS 忘记恢复现场或忘记剪枝

如果题目是“找路径”或“枚举方案”，就不只是普通遍历，还可能需要：

- 回溯
- 剪枝
- 恢复状态

### 3.4 有向图和无向图搞混

无向图通常要双向建边：

```python
graph[u].append(v)
graph[v].append(u)
```

有向图不能这么写。

## 5. Hot 100 经典例题

## 5.1 200. 岛屿数量

题意：二维网格中，统计由 `'1'` 组成的连通块数量。

### 思路

把每个陆地格子看成图中的一个节点：

- 上下左右相邻的陆地彼此连通
- 每找到一个还没访问过的陆地，就说明发现了一个新岛屿
- 然后用 `DFS` 或 `BFS` 把整个岛屿淹掉

### 代码

```python
def numIslands(grid):
    m, n = len(grid), len(grid[0])

    def dfs(i, j):
        if i < 0 or i >= m or j < 0 or j >= n:
            return
        if grid[i][j] != "1":
            return

        grid[i][j] = "0"

        dfs(i + 1, j)
        dfs(i - 1, j)
        dfs(i, j + 1)
        dfs(i, j - 1)

    ans = 0

    for i in range(m):
        for j in range(n):
            if grid[i][j] == "1":
                ans += 1
                dfs(i, j)

    return ans
```

### 这题训练的能力

- 把网格问题看成图搜索
- 掌握连通块统计

## 5.2 207. 课程表

题意：给定课程依赖关系，判断是否可以学完所有课程。

### 思路

这题本质是判断有向图中是否有环。

如果一门课有前置依赖，就说明：

- 必须先学前面的课程
- 这是一条有向边

做法是拓扑排序：

- 先把所有入度为 `0` 的节点入队
- 每次弹出一个节点，删除它的出边
- 如果最后能弹出所有节点，说明无环

### 代码

```python
from collections import deque


def canFinish(numCourses, prerequisites):
    graph = [[] for _ in range(numCourses)]
    indegree = [0] * numCourses

    for a, b in prerequisites:
        graph[b].append(a)
        indegree[a] += 1

    queue = deque(i for i in range(numCourses) if indegree[i] == 0)
    count = 0

    while queue:
        cur = queue.popleft()
        count += 1

        for nei in graph[cur]:
            indegree[nei] -= 1
            if indegree[nei] == 0:
                queue.append(nei)

    return count == numCourses
```

### 这题训练的能力

- 把依赖关系建成有向图
- 理解拓扑排序与判环

## 5.3 208. 实现 Trie（前缀树）

这题严格说更偏数据结构，但也经常和字符串图式搜索一起出现。

### 思路

Trie 的核心是：

- 每个节点表示一个前缀状态
- 从根到某个节点的路径，就是一个字符串前缀

它适合处理：

- 前缀匹配
- 字典搜索
- 单词搜索剪枝

### 代码

```python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False


class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for ch in word:
            if ch not in node.children:
                node.children[ch] = TrieNode()
            node = node.children[ch]
        node.is_end = True

    def search(self, word):
        node = self.root
        for ch in word:
            if ch not in node.children:
                return False
            node = node.children[ch]
        return node.is_end

    def startsWith(self, prefix):
        node = self.root
        for ch in prefix:
            if ch not in node.children:
                return False
            node = node.children[ch]
        return True
```

### 这题训练的能力

- 理解树形状态机
- 为 `79. 单词搜索`、字典树剪枝打基础
