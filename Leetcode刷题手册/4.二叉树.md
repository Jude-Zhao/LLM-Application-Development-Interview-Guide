# 二叉树

## 1. 核心思想

二叉树题最核心的思维方式有两种：

- **遍历**：把每个节点按某种顺序访问一遍
- **递归定义子问题**：一棵树的问题，拆成左右子树的问题

很多树题看起来不一样，但本质都在问：

- 当前节点该做什么？
- 左子树返回什么信息？
- 右子树返回什么信息？

所以树题最常见的写法就是递归。

## 2. 基础模板

### 2.1 前序遍历模板

前序顺序：

- 根
- 左
- 右

```python
def preorder(root):
    if not root:
        return

    处理当前节点
    preorder(root.left)
    preorder(root.right)
```

### 2.2 中序遍历模板

中序顺序：

- 左
- 根
- 右

```python
def inorder(root):
    if not root:
        return

    inorder(root.left)
    处理当前节点
    inorder(root.right)
```

### 2.3 后序遍历模板

后序顺序：

- 左
- 右
- 根

```python
def postorder(root):
    if not root:
        return

    postorder(root.left)
    postorder(root.right)
    处理当前节点
```

### 2.4 层序遍历模板

```python
from collections import deque


def levelOrder(root):
    if not root:
        return []

    res = []
    queue = deque([root])

    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)

            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)

        res.append(level)

    return res
```

## 4. 写二叉树题时最容易错的点

### 4.1 没想清楚递归函数的定义

例如递归函数到底返回：

- 一个数值
- 一个节点
- 一个布尔值
- 一条路径

这一步一旦定义清楚，代码通常就顺了。

### 4.2 混淆“当前节点该做什么”和“子问题返回什么”

很多树题的关键在于：

- 当前节点只负责整合左右子树的信息
- 左右子树各自递归求解

### 4.3 忘记空节点处理

树题的边界几乎永远是：

```python
if not root:
    return ...
```

## 5. Hot 100 经典例题

## 5.1 94. 二叉树的中序遍历

题意：返回二叉树的中序遍历结果。

### 思路

中序遍历顺序固定为：

- 左子树
- 当前节点
- 右子树

### 代码

```python
def inorderTraversal(root):
    res = []

    def dfs(node):
        if not node:
            return
        dfs(node.left)
        res.append(node.val)
        dfs(node.right)

    dfs(root)
    return res
```

### 这题训练的能力

- 熟悉最基础的树遍历
- 为后续 BST 题打基础

## 5.2 102. 二叉树的层序遍历

题意：按层从上到下返回节点值。

### 思路

使用队列做 BFS：

- 每次先记录当前层节点数量
- 再把这一层全部弹出并加入下一层节点

### 代码

```python
from collections import deque


def levelOrder(root):
    if not root:
        return []

    res = []
    queue = deque([root])

    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        res.append(level)

    return res
```

### 这题训练的能力

- 掌握 BFS 层序遍历
- 理解“队列长度就是当前层节点数”

## 5.3 104. 二叉树的最大深度

题意：求树的最大深度。

### 思路

定义递归函数返回：

- 以当前节点为根的子树最大深度

那么：

- 空节点深度为 `0`
- 当前节点深度 = `max(左子树深度, 右子树深度) + 1`

### 代码

```python
def maxDepth(root):
    if not root:
        return 0

    return max(maxDepth(root.left), maxDepth(root.right)) + 1
```

### 这题训练的能力

- 理解“树高 = 子树高 + 1”的递归定义
- 学会让递归函数返回信息

## 5.4 98. 验证二叉搜索树

题意：判断一棵树是否为合法 BST。

### 思路

BST 的关键性质：

- 中序遍历结果严格递增

所以只要中序遍历时，检查当前值是否大于前一个值即可。

### 代码

```python
def isValidBST(root):
    prev = None

    def inorder(node):
        nonlocal prev
        if not node:
            return True

        if not inorder(node.left):
            return False

        if prev is not None and node.val <= prev:
            return False
        prev = node.val

        return inorder(node.right)

    return inorder(root)
```

### 这题训练的能力

- 理解 BST 和中序遍历的关系
- 学会在遍历过程中维护额外状态

## 5.5 226. 翻转二叉树

题意：交换每个节点的左右子树。

### 思路

每到一个节点，就交换它的左右孩子，然后递归处理左右子树即可。

### 代码

```python
def invertTree(root):
    if not root:
        return None

    root.left, root.right = root.right, root.left
    invertTree(root.left)
    invertTree(root.right)
    return root
```

### 这题训练的能力

- 熟悉树的修改类递归
- 理解“先处理当前节点，再递归”这一类前序写法

## 5.6 236. 二叉树的最近公共祖先

题意：给定两个节点 `p` 和 `q`，求它们的最近公共祖先。

### 思路

递归函数含义：

- 在当前子树中寻找 `p` 和 `q`

如果：

- 左子树找到了一个
- 右子树找到了一个

那么当前节点就是最近公共祖先。

### 代码

```python
def lowestCommonAncestor(root, p, q):
    if not root or root == p or root == q:
        return root

    left = lowestCommonAncestor(root.left, p, q)
    right = lowestCommonAncestor(root.right, p, q)

    if left and right:
        return root
    return left if left else right
```

### 这题训练的能力

- 理解“左右子树分别汇报结果”的递归模型
- 掌握树上信息回传问题

## 5.7 114. 二叉树展开为链表

题意：把二叉树原地展开为先序遍历顺序的链表。

### 思路

这题更稳定的写法是“反向先序遍历”：

- 按 `右 -> 左 -> 根` 的顺序遍历
- 用一个 `prev` 指针记录上一个已经展开好的节点
- 当前节点的 `right` 指向 `prev`，`left` 置空

这样可以原地完成展开，时间复杂度是 `O(n)`。

### 代码

```python
def flatten(root):
    prev = None

    def dfs(node):
        nonlocal prev
        if not node:
            return

        dfs(node.right)
        dfs(node.left)

        node.right = prev
        node.left = None
        prev = node

    dfs(root)
```

### 这题训练的能力

- 理解树结构修改题
- 练习把递归结果重新拼接

## 7. 一句话总结二叉树

做树题时，优先问自己四个问题：

- 这题是在遍历，还是在让子树返回信息？
- 递归函数的定义到底是什么？
- 当前节点需要整合左右子树哪些结果？
- 空节点的返回值应该是什么？

这四个问题想清楚，绝大多数二叉树题都能拆开。
