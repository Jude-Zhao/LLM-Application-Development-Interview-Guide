# 栈

## 1. 核心思想

栈（Stack）是一种：

- 后进先出（LIFO）的数据结构
- 适合处理“最近的未处理状态”

最常见的两类栈题是：

- 普通栈：括号匹配、表达式解析、路径还原
- 单调栈：找左边 / 右边第一个更大或更小的元素

很多题虽然表面不同，但本质上都是：

> 当前元素，需要和之前“最近且仍然有效”的元素发生关系。

这时候就要想到栈。

---

## 2. 基础模板

### 2.1 普通栈模板

```python
def stack_template(nums):
    stack = []

    for x in nums:
        while stack and 不满足条件:
            stack.pop()

        stack.append(x)

    return stack
```

### 2.2 括号匹配模板

```python
def isValid(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}

    for ch in s:
        if ch in '([{':
            stack.append(ch)
        else:
            if not stack or stack[-1] != pairs[ch]:
                return False
            stack.pop()

    return not stack
```

### 2.3 单调栈模板

```python
def monotonic_stack(nums):
    stack = []
    res = [-1] * len(nums)

    for i, num in enumerate(nums):
        while stack and nums[stack[-1]] < num:
            idx = stack.pop()
            res[idx] = num
        stack.append(i)

    return res
```

适用场景：

- 下一个更大元素
- 下一个更小元素
- 柱状图面积
- 温度、股票跨度等“最近更大 / 更小”问题

---

## 3. 什么时候想到栈

### 3.1 需要匹配成对结构

例如：

- 括号是否合法
- 字符串解码
- 表达式求值

### 3.2 需要维护“最近的一个满足条件的位置”

例如：

- 下一个更大元素
- 上一个更小元素
- 最近的未匹配括号

### 3.3 当前元素会让前面某些元素“失效”

这是单调栈的典型特征。

如果当前元素一来，前面栈顶元素就可以确定答案并出栈，那通常就是单调栈。

---

## 4. 写栈题时最容易错的点

### 4.1 栈里存值还是存下标没想清楚

如果题目需要：

- 算距离
- 回到原数组取值
- 记录位置关系

通常要存**下标**，而不是只存值。

### 4.2 单调方向写反

单调栈常见两种：

- 单调递减栈：用来找下一个更大元素
- 单调递增栈：用来找下一个更小元素

方向写反，答案就全反了。

### 4.3 忘记处理边界

例如：

- 栈为空时不能直接访问 `stack[-1]`
- 柱状图问题通常要补一个哨兵

## 5. Hot 100 经典例题

## 5.1 20. 有效的括号

题意：判断括号字符串是否合法。

### 思路

遇到左括号就入栈，遇到右括号就和栈顶配对：

- 能配对就弹出
- 不能配对就返回 `False`

最后栈为空才合法。

### 代码

```python
def isValid(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}

    for ch in s:
        if ch in pairs.values():
            stack.append(ch)
        else:
            if not stack or stack[-1] != pairs[ch]:
                return False
            stack.pop()

    return not stack
```

### 这题训练的能力

- 掌握最基础的栈匹配
- 理解“后进先出”与括号嵌套的对应关系

## 5.2 155. 最小栈

题意：设计一个栈，支持 `push`、`pop`、`top`，并能在 `O(1)` 时间得到最小值。

### 思路

维护两个栈：

- 一个正常栈存所有元素
- 一个辅助栈存到当前位置为止的最小值

这样每次取最小值，直接看辅助栈栈顶即可。

### 代码

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = []

    def push(self, val):
        self.stack.append(val)
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))

    def pop(self):
        self.stack.pop()
        self.min_stack.pop()

    def top(self):
        return self.stack[-1]

    def getMin(self):
        return self.min_stack[-1]
```

### 这题训练的能力

- 理解“主栈 + 辅助栈”的设计方式
- 学会让每次操作都保持 `O(1)`

## 5.3 394. 字符串解码

题意：把形如 `3[a2[c]]` 的字符串解码成 `accaccacc`。

### 思路

遇到 `[` 时，把当前数字和当前字符串入栈。

遇到 `]` 时：

- 弹出之前的重复次数
- 弹出之前的字符串前缀
- 把当前字符串重复后拼回去

### 代码

```python
def decodeString(s):
    stack = []
    cur_num = 0
    cur_str = ""

    for ch in s:
        if ch.isdigit():
            cur_num = cur_num * 10 + int(ch)
        elif ch == '[':
            stack.append((cur_str, cur_num))
            cur_str = ""
            cur_num = 0
        elif ch == ']':
            prev_str, repeat = stack.pop()
            cur_str = prev_str + cur_str * repeat
        else:
            cur_str += ch

    return cur_str
```

### 这题训练的能力

- 理解嵌套结构为什么适合用栈
- 学会把“未完成上下文”压入栈中

## 5.4 739. 每日温度

题意：对每天温度，求还要等几天才能等到更高温度。

### 思路

维护一个**单调递减栈**，栈里存下标。

当今天温度更高时，说明今天就是前面某些天的“下一个更高温度”，这些天可以出栈并计算答案。

### 代码

```python
def dailyTemperatures(temperatures):
    n = len(temperatures)
    res = [0] * n
    stack = []

    for i, temp in enumerate(temperatures):
        while stack and temperatures[stack[-1]] < temp:
            idx = stack.pop()
            res[idx] = i - idx
        stack.append(i)

    return res
```

### 这题训练的能力

- 掌握单调栈最经典的模型
- 理解为什么栈里要存下标

## 5.5 84. 柱状图中最大的矩形

题意：求柱状图中最大的矩形面积。

### 思路

对于每根柱子，关键是找到：

- 左边第一个比它矮的位置
- 右边第一个比它矮的位置

这样就能知道以它为高时，最大宽度是多少。

单调递增栈可以在一次遍历中完成这个过程。

### 代码

```python
def largestRectangleArea(heights):
    stack = []
    ans = 0
    heights = [0] + heights + [0]

    for i, h in enumerate(heights):
        while stack and heights[stack[-1]] > h:
            height = heights[stack.pop()]
            width = i - stack[-1] - 1
            ans = max(ans, height * width)
        stack.append(i)

    return ans
```

### 这题训练的能力

- 掌握“左右第一个更小元素”的思路
- 学会给数组补哨兵，简化边界处理

## 7. 一句话总结栈

做栈题时，优先问自己三个问题：

- 题目是不是在处理“最近的未解决状态”？
- 当前元素是不是会让前面某些元素可以结算？
- 我到底该在栈里存值，还是存下标？

想清楚这三点，绝大多数栈题都会变得清晰。
