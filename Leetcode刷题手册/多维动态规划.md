# 多维动态规划

## 一、算法简介

**多维动态规划**是动态规划中的一种重要类型，其状态需要用**两个或更多维度**来表示。这类问题通常涉及：
- 二维网格/矩阵中的路径问题
- 两个序列的匹配/比较问题
- 需要同时考虑多个约束条件的优化问题

### 核心思想

将复杂问题分解为重叠子问题，通过**状态转移方程**在不同维度之间进行递推，最终得到全局最优解。

---

## 二、解题思路与模板

### 1. 二维 DP 通用模板

```python
# 1. 定义 dp 数组
dp = [[0] * (n + 1) for _ in range(m + 1)]

# 2. 初始化边界条件
dp[0][0] = initial_value

# 3. 状态转移
for i in range(1, m + 1):
    for j in range(1, n + 1):
        dp[i][j] = transition(dp[i-1][j], dp[i][j-1], dp[i-1][j-1], ...)

# 4. 返回结果
return dp[m][n]
```

### 2. 常见状态转移类型

| 类型 | 转移方程 | 典型问题 |
|------|----------|----------|
| 路径类 | `dp[i][j] = min/max(dp[i-1][j], dp[i][j-1]) + cost` | 最小路径和、不同路径 |
| 匹配类 | `dp[i][j] = dp[i-1][j-1] + match(i,j)` | 最长公共子序列 |
| 选择类 | `dp[i][j] = max(dp[i-1][j], dp[i][j-1], dp[i-1][j-1] + val)` | 编辑距离 |

## 三、LeetCode Hot 100 例题

### 例题 1：不同路径（LeetCode 62）

**题目描述：**
一个机器人位于一个 m x n 网格的左上角，机器人每次只能向下或者向右移动一步，到达右下角共有多少条不同的路径？

**解题思路：**
- `dp[i][j]` 表示到达位置 (i, j) 的路径数
- 状态转移：`dp[i][j] = dp[i-1][j] + dp[i][j-1]`（从上方或左方到达）
- 边界条件：第一行和第一列都为 1

**代码实现：**
```python
def uniquePaths(m: int, n: int) -> int:
    dp = [[1] * n for _ in range(m)]
    
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    
    return dp[m-1][n-1]

# 空间优化版本
def uniquePathsOptimized(m: int, n: int) -> int:
    dp = [1] * n
    for _ in range(m - 1):
        for j in range(1, n):
            dp[j] += dp[j-1]
    return dp[n-1]
```

### 例题 2：最小路径和（LeetCode 64）

**题目描述：**
给定一个包含非负整数的 m x n 网格，找一条从左上角到右下角的路径，使得路径上的数字总和最小。每次只能向下或向右移动。

**解题思路：**
- `dp[i][j]` 表示到达位置 (i, j) 的最小路径和
- 状态转移：`dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]`
- 边界条件：第一行和第一列需要累加

**代码实现：**
```python
def minPathSum(grid: list[list[int]]) -> int:
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    
    dp[0][0] = grid[0][0]
    
    # 初始化第一行
    for j in range(1, n):
        dp[0][j] = dp[0][j-1] + grid[0][j]
    
    # 初始化第一列
    for i in range(1, m):
        dp[i][0] = dp[i-1][0] + grid[i][0]
    
    # 状态转移
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
    
    return dp[m-1][n-1]

# 空间优化版本（原地修改）
def minPathSumOptimized(grid: list[list[int]]) -> int:
    m, n = len(grid), len(grid[0])
    
    for i in range(1, m):
        grid[i][0] += grid[i-1][0]
    
    for j in range(1, n):
        grid[0][j] += grid[0][j-1]
    
    for i in range(1, m):
        for j in range(1, n):
            grid[i][j] += min(grid[i-1][j], grid[i][j-1])
    
    return grid[m-1][n-1]
```

### 例题 3：打家劫舍 II（LeetCode 213）

**题目描述：**
房屋围成一圈，第一个和最后一个相邻。不能偷相邻的房屋，求能偷到的最大金额。

**解题思路：**
- 环形结构转化为两个线性问题：
  1. 不偷最后一间：范围 [0, n-2]
  2. 不偷第一间：范围 [1, n-1]
- `dp[i]` 表示前 i 间房能偷到的最大金额
- 状态转移：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`

**代码实现：**
```python
def rob(nums: list[int]) -> int:
    def rob_linear(houses: list[int]) -> int:
        n = len(houses)
        if n == 0:
            return 0
        if n == 1:
            return houses[0]
        
        dp = [0] * n
        dp[0] = houses[0]
        dp[1] = max(houses[0], houses[1])
        
        for i in range(2, n):
            dp[i] = max(dp[i-1], dp[i-2] + houses[i])
        
        return dp[n-1]
    
    n = len(nums)
    if n == 1:
        return nums[0]
    
    # 两种情况取最大值
    return max(rob_linear(nums[:-1]), rob_linear(nums[1:]))

# 空间优化版本
def robOptimized(nums: list[int]) -> int:
    def rob_linear(houses: list[int]) -> int:
        prev2, prev1 = 0, 0
        for num in houses:
            curr = max(prev1, prev2 + num)
            prev2, prev1 = prev1, curr
        return prev1
    
    n = len(nums)
    if n == 1:
        return nums[0]
    
    return max(rob_linear(nums[:-1]), rob_linear(nums[1:]))
```

### 例题 4：最长回文子串（LeetCode 5）

**题目描述：**
给定一个字符串 s，找到 s 中最长的回文子串。

**解题思路：**
- `dp[i][j]` 表示 s[i:j+1] 是否为回文串
- 状态转移：
  - 如果 s[i] == s[j] 且 j-i < 3，则是回文
  - 如果 s[i] == s[j] 且 dp[i+1][j-1] 为真，则是回文
- 记录最长回文的起始位置和长度

**代码实现：**
```python
def longestPalindrome(s: str) -> str:
    n = len(s)
    if n < 2:
        return s
    
    dp = [[False] * n for _ in range(n)]
    start, max_len = 0, 1
    
    # 单个字符都是回文
    for i in range(n):
        dp[i][i] = True
    
    # 填充 dp 表
    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            
            if s[i] == s[j]:
                if length == 2:
                    dp[i][j] = True
                else:
                    dp[i][j] = dp[i+1][j-1]
            
            if dp[i][j] and length > max_len:
                start = i
                max_len = length
    
    return s[start:start + max_len]
```

### 例题 5：编辑距离（LeetCode 72）

**题目描述：**
给定两个单词 word1 和 word2，计算将 word1 转换成 word2 所使用的最少操作数。操作包括：插入、删除、替换一个字符。

**解题思路：**
- `dp[i][j]` 表示 word1 前 i 个字符转换成 word2 前 j 个字符的最少操作数
- 状态转移：
  - 如果 word1[i-1] == word2[j-1]：`dp[i][j] = dp[i-1][j-1]`
  - 否则：`dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`

**代码实现：**
```python
def minDistance(word1: str, word2: str) -> int:
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 初始化边界
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    
    # 状态转移
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(
                    dp[i-1][j],
                    dp[i][j-1],
                    dp[i-1][j-1]
                ) + 1
    
    return dp[m][n]
```

## 四、总结与技巧

### 解题步骤
1. **定义状态**：明确 dp 数组每个维度的含义
2. **确定转移方程**：找出状态之间的递推关系
3. **初始化边界**：处理边缘情况
4. **计算顺序**：确保计算时依赖的状态已计算
5. **空间优化**：考虑是否可以用滚动数组或一维数组优化

### 常见优化技巧
- **滚动数组**：当只依赖前一行/列时，可将空间从 O(mn) 优化到 O(n)
- **原地修改**：如果允许修改输入数组，可直接在输入上操作
- **状态压缩**：某些问题可将二维状态压缩为一维

### 易错点
- 边界条件初始化不完整
- 数组越界（注意 i-1, j-1 的合法性）
- 状态转移方程推导错误
- 计算顺序导致依赖的状态未计算
