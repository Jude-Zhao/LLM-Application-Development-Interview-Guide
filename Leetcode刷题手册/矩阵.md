# 矩阵

## 1. 核心思想

矩阵题的本质通常不是“数学矩阵”，而是：

- 在二维数组里做遍历
- 处理行列关系
- 利用二维坐标进行搜索、模拟、标记

常见问题包括：

- 按顺序遍历二维数组
- 在二维空间做搜索
- 原地修改矩阵
- 根据行和列的特性优化查找

矩阵题最核心的能力是：

> 把一维数组的思考，扩展到“行 + 列 + 方向”三个维度。

def traverse(matrix):
    m, n = len(matrix), len(matrix[0])

    for i in range(m):

### 2.2 四方向遍历模板

```python
def neighbors(matrix, i, j):
    m, n = len(matrix), len(matrix[0])

    for dx, dy in [(1, 0), (-1, 0), (0, 1), (0, -1)]:
        x, y = i + dx, j + dy
        if 0 <= x < m and 0 <= y < n:
            处理 matrix[x][y]
```

def mark(matrix):

### 2.4 螺旋遍历模板

```python
def spiralOrder(matrix):
    top, bottom = 0, len(matrix) - 1
    left, right = 0, len(matrix[0]) - 1
    ans = []

    while top <= bottom and left <= right:
        for j in range(left, right + 1):
            ans.append(matrix[top][j])
        top += 1

        for i in range(top, bottom + 1):
            ans.append(matrix[i][right])
        right -= 1

        if top <= bottom:
            for j in range(right, left - 1, -1):
                ans.append(matrix[bottom][j])
            bottom -= 1

        if left <= right:
            for i in range(bottom, top - 1, -1):
                ans.append(matrix[i][left])
            left += 1

    return ans
```

## 3. 写矩阵题时最容易错的点

### 3.1 行列下标写反

要始终记住：

- `matrix[i][j]`
- `i` 是行
- `j` 是列

很多 bug 都来自把 `m` 和 `n` 搞混。

### 3.2 边界更新漏掉

像螺旋矩阵、搜索边界这类题，经常需要维护：

- 上下边界
- 左右边界

每走完一圈都要正确收缩边界。

### 3.3 一边遍历一边直接修改

如果题目要求“根据原矩阵状态统一修改”，就不能边扫边改。

否则后面的判断会读到已经被你改过的值。

### 3.4 忘记处理单行 / 单列

例如螺旋遍历时：

- 如果只剩一行
- 或只剩一列

要避免重复遍历。

## 5. Hot 100 经典例题

## 5.1 54. 螺旋矩阵

题意：按顺时针顺序返回矩阵中的所有元素。

### 思路

维护四个边界：

- `top`
- `bottom`
- `left`
- `right`

每一轮按四条边走：

- 从左到右
- 从上到下
- 从右到左
- 从下到上

走完一条边，就收缩对应边界。

### 代码

```python
def spiralOrder(matrix):
    top, bottom = 0, len(matrix) - 1
    left, right = 0, len(matrix[0]) - 1
    ans = []

    while top <= bottom and left <= right:
        for j in range(left, right + 1):
            ans.append(matrix[top][j])
        top += 1

        for i in range(top, bottom + 1):
            ans.append(matrix[i][right])
        right -= 1

        if top <= bottom:
            for j in range(right, left - 1, -1):
                ans.append(matrix[bottom][j])
            bottom -= 1

        if left <= right:
            for i in range(bottom, top - 1, -1):
                ans.append(matrix[i][left])
            left += 1

    return ans
```

### 这题训练的能力

- 二维边界控制
- 模拟类题目的细节处理

## 5.2 48. 旋转图像

题意：把 `n x n` 矩阵顺时针旋转 `90` 度，要求原地修改。

### 思路

一个非常经典的做法是分两步：

1. 先按主对角线翻转
2. 再左右翻转每一行

例如：

- 对角线翻转：`matrix[i][j]` 和 `matrix[j][i]` 交换
- 左右翻转：每一行头尾交换

### 代码

```python
def rotate(matrix):
    n = len(matrix)

    for i in range(n):
        for j in range(i + 1, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

    for row in matrix:
        row.reverse()
```

### 这题训练的能力

- 把几何变换拆成基础操作
- 原地修改二维数组

## 5.3 73. 矩阵置零

题意：如果一个元素为 `0`，则将其所在行和列全部设为 `0`。

### 思路

最直接的做法是：

- 先记录哪些行、哪些列需要变成 `0`
- 再统一修改

进一步优化空间时，可以把：

- 第一行当作列标记
- 第一列当作行标记

### 代码

```python
def setZeroes(matrix):
    m, n = len(matrix), len(matrix[0])
    rows = set()
    cols = set()

    for i in range(m):
        for j in range(n):
            if matrix[i][j] == 0:
                rows.add(i)
                cols.add(j)

    for i in range(m):
        for j in range(n):
            if i in rows or j in cols:
                matrix[i][j] = 0
```

### 这题训练的能力

- 处理“统一修改”的时机
- 理解原地标记和额外标记的区别
