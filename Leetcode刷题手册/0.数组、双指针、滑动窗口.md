# 数组、双指针与滑动窗口

## 1. 核心思想

这三个章节放在一起，是因为它们在 Hot 100 里高度重叠。

- 普通数组题：考察遍历、前后缀、原地修改、区间处理
- 双指针：两个位置协同移动，减少重复枚举
- 滑动窗口：维护一个连续区间，并动态更新窗口状态

它们的共同点是：

> 都是在数组或字符串上维护“当前区间 / 当前状态”。

当你看到下面这些信号时，优先考虑这一组方法：

- 求连续子数组 / 子串
- 在有序数组里从两端逼近
- 原地删除、去重、合并
- 求某个条件下的最长 / 最短区间

## 3. 写这类题时最容易错的点

### 3.1 双指针移动依据不明确

双指针不是“左右各走一步”这么简单，而是要先明确：

- 为什么该移动左指针
- 为什么该移动右指针

如果移动规则没有单调依据，就不是标准双指针题。

### 3.2 滑动窗口没有维护好窗口状态

例如你在窗口里维护：

- 频次
- 当前和
- 不重复字符数量

那么右扩和左缩时都必须同步更新。

### 3.3 把“子序列”和“子数组 / 子串”混淆

- 子数组 / 子串：必须连续，适合滑动窗口
- 子序列：不要求连续，往往不是滑动窗口

### 3.4 原地修改时覆盖了还没读到的值

快慢指针题里，要想清楚：

- `fast` 负责读
- `slow` 负责写

顺序不能乱。

## 5. Hot 100 经典例题

## 5.1 11. 盛最多水的容器

题意：选择两条线，和 `x` 轴组成容器，求最大面积。

### 思路

使用左右双指针：

- 面积由短板决定
- 想让面积变大，就必须尝试移动短板

如果移动长板：

- 宽度一定变小
- 高度上限仍然受短板限制

所以每次移动较短的一边。

### 代码

```python
def maxArea(height):
    left, right = 0, len(height) - 1
    ans = 0

    while left < right:
        h = min(height[left], height[right])
        ans = max(ans, h * (right - left))

        if height[left] < height[right]:
            left += 1
        else:
            right -= 1

    return ans
```

### 这题训练的能力

- 理解相向双指针的单调性
- 学会证明“为什么移动短板”

## 5.2 15. 三数之和

题意：找出所有和为 `0` 且不重复的三元组。

### 思路

经典做法：

1. 先排序
2. 固定第一个数
3. 剩下两个数用双指针逼近

排序后就有了单调性：

- 和太小，左指针右移
- 和太大，右指针左移

同时要注意去重。

### 代码

```python
def threeSum(nums):
    nums.sort()
    n = len(nums)
    ans = []

    for i in range(n - 2):
        if i > 0 and nums[i] == nums[i - 1]:
            continue

        left, right = i + 1, n - 1

        while left < right:
            total = nums[i] + nums[left] + nums[right]

            if total == 0:
                ans.append([nums[i], nums[left], nums[right]])
                left += 1
                right -= 1

                while left < right and nums[left] == nums[left - 1]:
                    left += 1
                while left < right and nums[right] == nums[right + 1]:
                    right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1

    return ans
```

### 这题训练的能力

- 排序后构造双指针
- 处理重复答案

## 5.3 42. 接雨水

题意：给定柱子高度，求能接多少雨水。

### 思路

双指针做法的关键是维护：

- 左侧最高 `left_max`
- 右侧最高 `right_max`

如果 `left_max < right_max`，那么当前左边位置能接多少水，只由 `left_max` 决定。

### 代码

```python
def trap(height):
    left, right = 0, len(height) - 1
    left_max = right_max = 0
    ans = 0

    while left < right:
        left_max = max(left_max, height[left])
        right_max = max(right_max, height[right])

        if left_max < right_max:
            ans += left_max - height[left]
            left += 1
        else:
            ans += right_max - height[right]
            right -= 1

    return ans
```

### 这题训练的能力

- 用双指针替代前后缀数组
- 理解“哪一侧的答案已经可以确定”

## 5.4 283. 移动零

题意：把所有 `0` 移到数组末尾，同时保持非零元素相对顺序。

### 思路

使用快慢指针：

- `fast` 扫描所有元素
- `slow` 指向下一个该放非零元素的位置

先把所有非零元素稳定地往前写，再把后面补 `0`。

### 代码

```python
def moveZeroes(nums):
    slow = 0

    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow], nums[fast] = nums[fast], nums[slow]
            slow += 1
```

### 这题训练的能力

- 原地数组重排
- 快慢指针的读写分工

## 5.5 3. 无重复字符的最长子串

题意：返回不含重复字符的最长子串长度。

### 思路

这是最典型的滑动窗口题。

维护一个无重复窗口：

- `right` 负责扩张窗口
- 如果出现重复字符，就不断移动 `left` 缩小窗口
- 每次更新窗口最大长度

### 代码

```python
def lengthOfLongestSubstring(s):
    window = set()
    left = 0
    ans = 0

    for right, ch in enumerate(s):
        while ch in window:
            window.remove(s[left])
            left += 1

        window.add(ch)
        ans = max(ans, right - left + 1)

    return ans
```

### 这题训练的能力

- 掌握可变长度滑动窗口
- 理解“先扩张，违规后收缩”的基本套路
